# MySQL-notes

关于MySQL的学习笔记，阅读顺序分别是：

- 数据库基础：包括MySQL体系结构、SQL语言、多表查询、存储过程、触发器和视图等知识点
- 数据库进阶一：包括记录结构、数据页结构、表空间、数据目录和索引等知识点，这部分知识点比较底层，不是很好理解。
- 数据库进阶二：包括日志（redo、undo、bin的详解）、主从复制、事务、MVCC机制和锁等知识点。
- SQL优化
